# Enhanced KPI Dashboard - Enterprise Digital Nervous System

A comprehensive, interactive dashboard for supply chain management and enterprise analytics. This application provides real-time insights, simulation capabilities, and workflow management tools to optimize business performance.

## Features

### 1. Summary & Triage
-   **Dashboard Overview**: High-level view of Enterprise Health, OTIF, Stock Out Rate, and Cost Per Unit.
-   **Time-Based Views**: dedicated sub-views for Yearly, Quarterly, Monthly, and Weekly performance summaries.
-   **Critical Issues & Opportunities**: Prioritized lists of actionable insights with root cause analysis.
-   **Enterprise Copilot**: Chat interface for quick summaries and triage assistance.

### 2. Planning Agent
-   **Analytics Studio**: detailed KPI analysis with trend charts, contributors, and root cause focus.
-   **Simulation Studio**: Scenario planning tool to adjust variables (safety stock, lead time, etc.) and predict KPI outcomes.

### 3. Control Agent
-   **Action Tracker**: Kanban-style board for managing tasks and tickets.
-   **Workflow Studio**: Value Stream Map (VSM) visualization to identify bottlenecks in the order lifecycle.

### 4. Improvement Agent
-   **Metrics**: Track specific improvement initiatives and their impact.

### 5. News Insights
-   External market data and news relevance.

## Technical Details
-   **Frontend**: HTML5, CSS3, Vanilla JavaScript.
-   **Charts**: Chart.js for data visualization.
-   **Icons**: SVG icons.
-   **Design**: Responsive layout with a clean, modern aesthetic using CSS Grid and Flexbox.

## Usage
1.  Open `index.html` in a modern web browser.
2.  Navigate through the tabs to explore different agents and studios.
3.  Click on KPI cards or use the chat interfaces to interact with the data.
